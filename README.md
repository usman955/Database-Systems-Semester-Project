## Description
A web application designed to track and manage waste reporting and recycling efforts. It allows students to report issues, maintenance to track status, and admins to manage users and monitor recycling bins.

## Technologies Used
 Backend: Node.js, Express.js
 Frontend: EJS, Tailwind CSS
 Database: SQLite (better-sqlite3)

## Installation & Setup

1. Prerequisites: Make sure you have [Node.js](https://nodejs.org/) installed on your machine.
2. Install Dependencies: Open a terminal in the project folder and run:
   ```bash
   npm install
   ```
3. Initialize Database: Run the following command to set up the SQLite database and create the required tables:
   ```bash
   npm run init-db
   ```
4. Start the Application: 
   ```bash
   npm start
   ```
5. Open your browser and navigate to `http://localhost:3000` (or the port specified in your console).

## Note
The `node_modules` folder, local `.db` files, and user uploads are ignored in Git to keep the repository clean.
